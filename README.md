[ DESCRIPTION ]
------------------------------------------------------------
FNX-Host is a dedicated static hosting platform designed to 
serve custom scripts, assets, and project files. Built for 
speed and simplicity, it provides a direct access point for 
developers and users to interact with FNX-based tools and 
configurations.

------------------------------------------------------------
[ KEY FEATURES ]
------------------------------------------------------------
* High Availability: Hosted via GitHub Pages for 99.9% uptime.
* Glassmorphic UI: Modern, clean, and eye-friendly design.
* Lightweight: Minimal code for ultra-fast loading times.
* Script Hub: Centralized location for hosting LUA/Python 
  utility scripts.

------------------------------------------------------------
[ USAGE ]
------------------------------------------------------------
1. Access the host via the provided URL.
2. Navigate through the UI to find the specific script or 
   file required.
3. Copy the raw link or use the direct integration code 
   provided on the site.

------------------------------------------------------------
[ TECHNICAL STACK ]
------------------------------------------------------------
* Frontend: HTML5, CSS3 (Custom Styles)
* Deployment: GitHub Actions / Pages
* Compatibility: Optimized for Desktop and Mobile browsers.
